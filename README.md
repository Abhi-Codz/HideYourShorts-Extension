<h1 align="center" style="text-align: center; font-size: 35px; font-weight: 700;">HideYourShorts</h1>
<p align="center" style="text-align: center; font-size: 16px;">A toggling browser extension, for personal use rather, to bid shorts addiction a farewell.. Well, smart farewell.</p>

![](https://github.com/Abhi-Codz/HideYourShorts-Extension/blob/91d9559c146236727ea4dd96c9b4d4d03689e5c2/StatesOfHYD.png)

**HideYourShorts** is with certainty a lightweight browser extension designed to remove YouTube Shorts from the areas within YouTube that one may find distracting, whilst also preserving the rest of the YouTube experience.
<br>

### Features:

**I. Hide Shorts shelves and recommendations from the YouTube Home page**<br>
**II. Hide Shorts sections from YouTube Search Results**<br>
**III. Remove the Shorts shortcut from YouTube's navigation sidebar**<br>
**IV. Independent controls for each feature**<br>
**V. Global extension toggle for enabling or disabling all functionality instantly**<br>
**VI. Lightweight design with minimal resource usage**<br>

Platform: Chromium Based Browsers / Mozilla Firefox

## How to install, you may ask?

### Chrome / Chromium Browsers
I. Download, or clone this repository. <br>
```bash
git clone https://github.com/Abhi-Codz/HideYourShorts-Extension.git
```
II. Open: <br>
```bash
chrome://extensions
```
<br>
III. Enable Developer Mode. <br>
IV. Click Load unpacked. <br>
V. Select the HideYourShorts directory. <br>
VI. Open YouTube and enjoy a cleaner browsing experience. 

### Mozilla FireFox 
Available on Mozilla Add-ons (AMO)

```bash
https://addons.mozilla.org/en-US/firefox/addon/hideyourshorts/
```
